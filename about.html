Om mig

Hej! Jag heter Moody och jag studerar till mjukvaruutvecklare med fokus på mobilitettjänster. Den här webbplatsen skapade jag som en del av ett hobbyprojekt för att öva på HTML, CSS och JavaScript. Här delar jag med mig av mina projekt, bilder och små interaktiva demoexempel.

Tekniker jag använder på den hemsidan

  • HTML5: Semantisk struktur och tillgänglighet
  • CSS3: Flexbox, Grid, variabler och responsiv design
  • JavaScript (ES6): DOM-manipulation, moduler och events
  • LocalStorage: För att spara Todo-listan lokalt

Vad jag vill lära mig framöver

Jag vill fördjupa mig mer i JavaScript-ramverk som React och Vue, samt lära mig grunderna i backend med Node.js och API-hantering. Mitt mål är att kunna bygga fullständiga webbappar med både frontend och backend.
